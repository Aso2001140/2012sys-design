```uml
@startuml

opt 未登録
ユーザー -> サンプルサイト : ユーザー登録
サンプルサイト -> DBサーバー : ユーザー登録
DBサーバー -> DBサーバー : 登録処理
DBサーバー -> サンプルサイト : 登録結果

alt 登録成功
サンプルサイト -> ユーザー : 登録メッセージを表示
else 登録失敗
サンプルサイト -> ユーザー : 失敗メッセージを表示

end
end


ユーザー -> サンプルサイト : 検索（商品名）
サンプルサイト -> DBサーバー : 検索処理
DBサーバー -> DBサーバー : 検索処理
DBサーバー -> サンプルサイト : 検索結果
サンプルサイト -> ユーザー : 検索結果

ユーザー -> サンプルサイト : 商品を購入
サンプルサイト -> サンプルサイト : 合計の計算
サンプルサイト -> ユーザー : 合計金額の表示
@enduml
```
